【何をする関数？】
ボット全体で使うデータベースの初期設定をまとめて行う。
users テーブルの作成、カラム追加、進捗用テーブルの作成を一括して実行する。

【引数（入力）の説明】
db_file: データベースファイルのパス（例: "database.db"）

【戻り値（出力）の説明】
なし（None）。テーブルやカラムを作るだけで何も返さない。

【どんな時に使う？】
ボットを初めて起動するときや、データベースが新しく作られるときに呼び出すセットアップ関数。

【ポイント・注意点】
通常の関数（asyncではない）なので await は不要
"CREATE TABLE IF NOT EXISTS" を使っているのでテーブルが既存でもエラーにならない
"ALTER TABLE ... ADD COLUMN" は既にカラムがある場合は OperationalError になるが、try/except で無視する
内部で init_progress_tables も呼び出して進捗管理用テーブルも作成する
users テーブルのカラム: atcoder_name（主キー）, discord_name, discord_id, resister_id
